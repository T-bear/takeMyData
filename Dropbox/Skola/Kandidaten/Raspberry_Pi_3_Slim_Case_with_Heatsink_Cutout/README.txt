# Summary

Slim case for the new Raspberry Pi 3!  Case fits together with no mounting hardware.

Remix Improvements:
* Added cutout for a CPU heatsink since RPi3s run a little hot.  I used the following 13mm x 18mm heatsink  -> http://www.amazon.com/gp/product/B00ZZ9P3QC
* Tightened up a bunch of dimensions, camera & display ribbon cutouts shrunk so they're not so oversized
* Added vent on case bottom directly below CPU.  I used the following bumper feet -> http://www.amazon.com/gp/product/B008DVSFNM
* Increased case top and bottom faces from 1mm to 1.5mm to increase strength and reduce flex

I've uploaded two versions of both the case top and bottom so you can pick if you want display & camera cable cutouts or not, and if you want the vent cutouts on the bottom or not.

Enjoy!
